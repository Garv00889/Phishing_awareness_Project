# 🔐 Phishing Awareness Project

This is a PHP-based web application built for educational purposes to demonstrate phishing awareness and security concepts. It runs on a local server using XAMPP.

## ⚠️ Disclaimer

This project is created strictly for **educational and awareness purposes only**.

- It is designed to demonstrate how phishing attacks work
- It should NOT be used for any illegal, harmful, or unethical activities
- The developer does not support or encourage phishing or any cybercrime

By using this project, you agree that:
- You will use it only for learning and research
- You will not use it to target or deceive any real users or systems
- You are fully responsible for how you use this code

⚠️ Misuse of this project is strictly prohibited.
---

## 🚀 Features
- Phishing awareness simulation pages
- User dashboard
- Campaign management
- Login system
- Educational phishing examples

---

## 🧰 Requirements

To run this project, you need:

- XAMPP (Apache + MySQL)
- PHP (comes with XAMPP)
- Web browser (Chrome recommended)

Download XAMPP:
https://www.apachefriends.org/
